# NWPU-HC课程学习资料库

一款专门面向 **西北工业大学 教育实验学院 / 未来技术学院 / 学科交叉中心（HC）** 学子的民间开源课程资料合集。

## 1. 前言与初衷

 2026年5月24日，我通过朋友偶然间了解到：软件学院的学长牵头整理学院课程资料、为同学们提供帮助。受此民间优秀开源资料库的启发，我发现虽然学校有“启翔湖畔”等优秀的通识资料站，但 **HC 有很大一部分课程为学院单独开设、单独命题**。其考察范围、讲授深度与全校通识课存在一定差异。

为了打破信息差，帮助每一位在 HC 砥砺前行的同学，我决定建立这个专属于 HC 学院的资料仓库。一个优质的资源库无法靠一个人独立完成，它需要一代代 HC 学子们薪火相传、共同维护。欢迎未来的你积极参与进来，共同建设属于我们自己的荣誉家园！

## 3. 如何贡献资料

如果你手里有：**历年复习大纲、手写/电子版精美笔记、非涉密期末真题、优秀作业参考、实验开源代码**等课程学习资料，欢迎通过以下方式贡献：

**1.硬核提交**：Fork本仓库，将仓库放入对应文件夹后提交Pull Request。

**2.轻松分享**：如果不熟悉Git操作，可以直接将资料注明【年级-科目】发送到我的邮箱（*xibeisaodeseng@gmail dot com*），我会经过查看后，帮你上传。
