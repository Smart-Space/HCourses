# NWPU-HC课程学习资料库

面向 **西北工业大学 教育实验学院 / 未来技术学院 / 学科交叉中心（HC）** 学子的民间开源课程资料合集。

## 1. 前言与初衷

 长期以来，HC培养方案特殊，造成了课程复习资料（往年考题等）难以获取。虽然学校有“启翔湖畔”等优秀的通识资料站，但 **HC 有很大一部分课程为学院单独开设、单独命题**。其考察范围、讲授深度与全校通识课存在一定差异。面对类似问题，计算机学院、软件学院等均有建立共享资料库。2026年5月，为了打破信息差、增强交流，帮助每一位在 HC 砥砺前行的同学，我们尝试建立该资料站，初期聚焦于课程考题汇总与整理，也期待未来能够增加保研考研、实习就业、交换出国等模块。

> ⚖️ **免责声明**
> 本仓库收录的所有资料均来源于公开渠道或同学自愿分享，仅供学术交流与期末复习使用，严禁用于任何商业用途。
