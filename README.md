# NWPU-HC课程学习资料库

面向 西北工业大学 教育实验学院 / 未来技术学院 / 学科交叉中心（HC） 学子的民间开源课程资料合集。

## 1. 初衷

长期以来，HC培养方案特殊，造成了课程复习资料（往年考题等）难以获取。虽然学校有“启翔湖畔”等优秀的通识资料站，但 HC 有很大一部分课程为学院单独开设、单独命题。其考察范围、讲授深度与全校通识课存在一定差异。面对类似问题，计算机学院、软件学院等均有建立共享资料库。2026年5月，为了打破信息差、增强交流，帮助每一位在 HC 砥砺前行的同学，我们尝试建立该资料站，初期聚焦于课程考题汇总与整理，也期待未来能够增加保研考研、实习就业、交换出国等模块。

> ⚖️ 免责声明
>
> 本仓库收录的所有资料均来源于公开渠道或同学自愿分享，仅供学术交流与期末复习使用，严禁用于任何商业用途。
>
> 如果仓库中的任何内容侵犯了您的正当权益，请立即提交 Issue 或联系维护者（xibeisaodeseng@gmail dot com），我将在第一时间核实并删除。

## 3. 如何贡献资料

如果你手里有：

- 历年复习大纲
- 手写/电子版优质笔记
- 非涉密期末真题
- 优秀作业参考
- 实验开源代码
- 课程学习经验总结

欢迎通过以下方式贡献：

### 方式一：Pull Request

Fork 本仓库，将资料放入对应课程文件夹后提交 Pull Request。

### 方式二：邮件投稿

如果不熟悉 Git 操作，也欢迎直接将资料注明【年级-课程名称】发送至邮箱：

xibeisaodeseng@gmail.com

我们会在审核后统一整理上传。

## 4. 学习与发展资源

除了课程资料外，我们也推荐以下优秀学习资源，希望能够帮助 HC 同学拓宽视野、提升自主学习能力。

### CS自学指南（CS DIY）

计算机科学领域最知名的中文自学路线之一，涵盖课程学习、技术成长、科研与就业规划等内容。

🌐 https://csdiy.wiki/

---

### NPU2World

由西工大学长学姐维护的出国（境）交流信息平台，收录大量交换项目、暑校、联合培养与申请经验。

🌐 https://npu2world.github.io/NPU2World/

---

### The Missing Semester

MIT 开设的经典课程《计算机教育中缺失的一课》。

课程内容涵盖：

- Linux
- Shell
- Git
- SSH
- Vim
- 自动化脚本
- 开发环境配置

这些内容往往不会在正式课程中系统讲授，却是科研与工程实践中的基础技能。

🌐 https://missing-semester-cn.github.io/
